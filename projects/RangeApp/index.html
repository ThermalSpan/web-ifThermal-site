RangeApp

Explanation

This processing sketch was built as a project for "Introduction to Computational Geometry." It demonstrates the use of a kd-tree in performing a range search. The user can place place points and select points using rectangle or cicle ranges. There are also a variety of extra visual effects available to demonstrate how the tree works.

Controls

  • j - Toggle drawing subtree boundaries.
  • k - Toggle drawing tree relations.
  • l - Toggle drawing of tree regions explored during selection. (Except the root, that would be the whole thing.)
  • e - Toggle drawing of gnoman.
  • r - Add one random point
  • t - Add twenty random points.
  • c - Clear points and tree.
  • Right click drag for zoom.
